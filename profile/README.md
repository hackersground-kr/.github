## 안녕하세요 👋 해커그라운드입니다!

해커그라운드는 대구/경북 지역 최고의 독립 개발자 커뮤니티입니다.

해커그라운드는 대구/경북 최고의 개발자 커뮤니티 육성을 통해 대구/경북지역 개발자의 자립을 목표로 합니다.
